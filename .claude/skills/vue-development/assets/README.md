# Assets Directory

This directory contains files and templates that are used as boilerplate when creating new Vue projects or components. These assets are **not** loaded into context but are copied or referenced when generating output.

## Contents

### vue-starter-project/
A minimal Vite + Vue 3 + TypeScript starter project demonstrating:
- Feature-based architecture
- Testing setup (Vitest + Testing Library)
- Pinia store with Composition API
- Example composable
- Type-safe component patterns

## Usage

When users ask to create a new Vue project or scaffold a feature, reference these templates as starting points. The boilerplate follows all best practices documented in the skill's references/.
